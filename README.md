# 运行环境
release中打包的.exe程序在Windows 10操作系统运行 或者可以把项目导入 Cocos Creator V2.2.2中在Main场景下运行

# 编译方法
将项目导入到Cocos Creator V2.2.2中编译运行

# 使用方法
在主界面中可以选择人人对战、人机对战、在线对战。在在线对战中需要使用校园网进行登录，在线对战的房间场景中可以选择开始或者AI开始
